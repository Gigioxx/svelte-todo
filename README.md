# Todo App

## Description

This is a simple todo app that allows you to add, and delete tasks. It also allows you to mark tasks as complete.

## Tech Stack

- Svelte
- SvelteKit
- TailwindCSS
- Typescript

## Installation

1. Clone the repo
```bash
git clone https://github.com/Gigioxx/svelte-todo.git
```

2. Install dependencies
```bash
bun i
```

3. Run the app
```bash
bun dev
```
